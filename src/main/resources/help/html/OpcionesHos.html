"Bienvenidos a la aplicacion de gestion hospitalaria"


Esta ventana contiene 5 botones , los 4 primeros accesibles para cualquier medico y el de creacion de hospital solo para el Administrador del sistema

Al pulsar el boton se abrira una nueva ventana que permitira gestionar diferentes cosas

La aplicacion permite:

  • Dar de alta informes
  • Asignar citas
  • Gestiones hospitalarias
  • Eliminar medico/paciente
  • Gestiones medicas
  • Crear Hospital
